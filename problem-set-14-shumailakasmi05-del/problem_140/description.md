## Problem: Calculate Median

### Description
Write a program that reads a list of space-separated integers and prints the median value. The median is the middle value when the numbers are sorted. If there are an even number of values, the median is the average of the two middle numbers. The result must be printed with 2 decimal places.

### Input Format
Single line containing space-separated integers

### Output Format
Print the median value formatted to 2 decimal places

### Examples

---
Input:
```
1 2 3 4 5
```
Output:
```
3.00
```

---
Input:
```
1 2 3 4
```
Output:
```
2.50
```

---
Input:
```
42
```
Output:
```
42.00
```

---
Input:
```
5 1 9 3 7
```
Output:
```
5.00
```

---
Input:
```
-5 -1 0 3 7
```
Output:
```
0.00
```
