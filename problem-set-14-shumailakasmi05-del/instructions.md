#### Python Usage and Restrictions

**You may use:**
- Dictionaries (basic operations: `dict[key]`, `dict[key] = value`, `for key in dict:`)
- Lists and list operations
- For loops
- Range function
- String/list slicing
- Lambda expressions
- Functions as arguments (`sorted(key=...)`, `map()`, `filter()`, `reduce()`, etc.)

**Do not use:**
- Regular expressions (`re` module)
- Exception handling (`try`/`except`/`finally`)
- Built-in helpers: `any()`, `all()`, `enumerate()`, `zip()`, `bin()`, `format()`, `zfill()`
- List/string aggregation functions: `min()`, `max()`, `sum()` on lists
- Dictionary iteration methods: `.items()`, `.keys()`, `.values()`
- Comprehensions (list/dict/set)
- Sets and set operations
- Generators
