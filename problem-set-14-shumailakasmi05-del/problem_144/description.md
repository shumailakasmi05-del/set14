## Problem: Sort by Squared Values

### Description
Write a program that reads a list of integers and sorts them by their squared values in ascending order. If two integers have the same squared value, sort them by their original value in ascending order.

### Input Format
Single line containing space-separated integers

### Output Format
Print the sorted integers separated by a single space

### Examples

---
Input:
```
3 -2 5 1 -4
```
Output:
```
1 -2 3 -4 5
```

---
Input:
```
-3 3 -1 1 0
```
Output:
```
0 -1 1 -3 3
```

---
Input:
```
-5
```
Output:
```
-5
```

---
Input:
```
-10 -5 0 5 10
```
Output:
```
0 -5 5 -10 10
```

---
Input:
```
4 -4 2 -2 0
```
Output:
```
0 -2 2 -4 4
```
