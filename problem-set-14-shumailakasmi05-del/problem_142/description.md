## Problem: Reverse-Order Sort

### Description
Write a program that reads a list of lowercase strings and sorts them based on their reversed form. When comparing strings for sorting, each string should be read from right to left (reversed), but the original strings should be printed in the output.

### Input Format
Single line containing space-separated lowercase strings

### Output Format
Print the strings sorted by their reversed forms, separated by a single space

### Examples

---
Input:
```
abc dog cat
```
Output:
```
abc dog cat
```

---
Input:
```
bat tab rat
```
Output:
```
tab bat rat
```

---
Input:
```
apple banana cherry
```
Output:
```
banana apple cherry
```

---
Input:
```
hello
```
Output:
```
hello
```

---
Input:
```
aa ab ba bb
```
Output:
```
aa ba ab bb
```
