## Problem: Count One Bits

### Description
Write a program that reads an integer and prints the number of 1 bits in its binary representation. For example, 5 in binary is 101 (two 1 bits), 7 in binary is 111 (three 1 bits), and 8 in binary is 1000 (one 1 bit).

### Input Format
Single line containing a non-negative integer

### Output Format
Print a single integer representing the count of 1 bits

### Examples

---
Input:
```
5
```
Output:
```
2
```

---
Input:
```
7
```
Output:
```
3
```

---
Input:
```
0
```
Output:
```
0
```

---
Input:
```
8
```
Output:
```
1
```

---
Input:
```
15
```
Output:
```
4
```

---
Input:
```
1023
```
Output:
```
10
```
